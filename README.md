# Indian Jobs AI Exposure

Interactive visualization of AI exposure across India's workforce, combining organized-sector job listings with PLFS 2023–24 informal-sector estimates.

## Files
- `index.html` — interactive treemap + charts
- `job_data.json` — processed visualization data
- `preprocess.py` — original organized-sector preprocessing
- `preprocess_expanded.py` — expanded preprocessing with PLFS estimates
